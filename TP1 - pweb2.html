Ojos animatrónicos

Carlos (el apodo que le pusimos) fue el primer proyecto de nuestro emprendimiento. La idea era utilizar nuestros conocimientos de servo motores para crear un mecanismo microcontrolado que pudiera imitar los movimientos de los ojos de animatrónicos reales. La idea de este proyecto esta fuertemente inspirada en los personajes de la popular saga de videojuegos "Five Nights At Freddy's". Haz click aquí para ver más.

R.A.D.: Registro de Asistencia Digital

El sistema de Registro de Asistencia Digital fue el proyecto que nos llevó hasta la Feria de Ciencias de Merlo en el año 2024, la cual fue nuestra primera competencia. Este consistia en un sistema de identificación por radiofrecuencia (RFID) integrado a una base de datos para registrar la entrada y salida de alumnos, mejorando la seguridad y garantizando una respuesta rápida en caso de desaparición. Dron Submarino (work in progress)

Nuestro proyecto más reciente, inspirado de un videojuego llamado Subnautica, se trata de un dron submarino con la capacidad de escanear el lecho marino donde este se encuentra. El dron sera capaz de ser controlado por el usuario mediante un software especializado, atravez de tecnología BlueTooth(se esta evaluando la posibilidad de usar una pagina HTML). Este dron esta siendo diseñado para ser multiproposito, por lo que será apto para aplicaciones cientificas, patrulla, seguridad o incluso para espionaje!
